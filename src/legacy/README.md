# Legacy del TFG (archivado, no pipeline activo)

Este directorio conserva material historico para memoria y trazabilidad, pero
queda desacoplado del flujo principal de ejecucion.

## Estructura

- `01_color_detection_demo/`: detector HSV antiguo y pruebas iniciales.
- `02_slider_controller_tests/`: control manual con sliders (`joint_state_publisher_gui`).
- `03_gazebo_segfault_bisect/`: worlds y pruebas usadas para aislar segfaults.
- `04_old_worlds/`: worlds antiguos o con nombre confuso (incluye el antiguo `empty.world`).
- `05_archived_nodes/`: nodos reemplazados por la arquitectura modular actual.
- `06_old_launchers/`: launch files de demos/pipelines anteriores.

Nada de este contenido se elimina; solo se aparta del camino principal.
